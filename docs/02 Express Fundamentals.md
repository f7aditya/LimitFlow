# Express Fundamentals

## What is Express?

Express is a minimal and flexible web framework built on top of Node.js.

It simplifies backend development by providing features like routing, middleware, request handling, and response handling.

---

## Why do we use Express?

Without Express, we would have to build everything using Node's HTTP module.

Express reduces boilerplate code and makes backend development easier.

---

## What can Express do?

- Create servers
- Build REST APIs
- Handle routes
- Use middleware
- Send responses
- Handle errors

## Key Point

Express is built on top of Node.js.

Node.js provides the runtime.

Express provides the framework.
